# Programa4

Este programa se vincula a 3 bases de datos distintas y consta de 3 ejercicios:

*Ejercicio1: Selector de lugar de salida y de destino para viajes, segun los lugares de la base de datos "Viajes".

*Ejercicio2: Carga una tabla con los contenidos de la base de datos "Neptuno" y permite filtrar el contenido que se quiere ver.

*Ejercicio3: Selecciona un tema de la base de datos "Libreria" y muestra los libros de ese tema seleccionado.
